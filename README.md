# rescar
### 本專題旨在研究與開發一套結合點餐系統和送餐機器人的智慧餐飲服務解決方案。隨著科技的不斷進步，餐飲業也需要不斷創新以滿足消費者的需求。傳統的點餐送餐方式存在著效率低下、人力成本高昂和溝通不順等問題。因此，我們的專題旨在利用科技的力量改進現有的餐飲服務流程，提供更快速、更便捷的點餐和送餐體驗。
